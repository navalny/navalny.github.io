Локальный Навальный

Скачать

Локальное зеркало блога Алексея Навального. Включает все посты с самого первого. Сделано с помощью Jekyll, Lanyon и lj2jekyll.

Как собрать?

  1. Установите Jekyll.
  2. Скачайте архив по ссылке выше или через git, выполнив команду git clone https://github.com/navalny/localnavalny.
  3. Перейдите в папку с локальным Навальным и запустите jekyll build. В папке _site появится статичное локальное зеркало блога Навального.
  4. Можете просматривать его через браузер или разместить на своем сервере.

Новые посты в репозиторий добавляются автоматически. Для закачки обновлений запустите git pull и пересоберите блог с помощью jekyll build.
